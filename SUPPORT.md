## How to Ask for Help

To get the fastest response:

1. **Search first**: Check existing [discussions](https://github.com/neural-garage/bury/discussions) and [issues](https://github.com/neural-garage/bury/issues)
2. **Be specific**: Include version numbers, command output, and error messages
3. **Provide context**: 
   - What are you trying to do?
   - What did you expect to happen?
   - What actually happened?
4. **Share code**: Use code blocks or gists for examples

### Good Example

```markdown
**Tool**: bury v0.1.0
**OS**: macOS 14.0
**Problem**: Getting false positives for exported functions

I'm running: `bury --format json ./src`

Expected: Exported functions should not be marked as dead
Actual: `export function foo()` is marked as dead code

Code example: ...
```
